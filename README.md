Screener
========

A screen server emulator

To Run
------

```bash
python screener/app.py
```

Test Client
-----------

```bash
python screener/client.py
```

Unit Tests
----------

Install nose and use:

```bash
nosetests
```

Documentation
-------------

Install Sphinx and use:

```bash
cd docs
make html
```

You'll find the docs under `docs/_build`.
